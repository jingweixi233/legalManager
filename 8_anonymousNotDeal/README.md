## 裁判文书公开时应匿名处理的信息未匿名处理

输入

案件裁判文书

输出

判断裁判文书中是否有应匿名处理的信息

评查要求

判断准确率不低于80%

支持不少于5类司法文书数据格式

评查规则

婚姻家庭、继承纠纷案件中的当事人及其法定代理人；刑事案件被害人及其法定代理人、附带民事诉讼原告人及其法定代理人、证人、鉴定人；未成年人及其法定代理人
建议只做加粗部分检测
