案件裁判文书

输出

判断裁判文书公开是否合法


评查要求

判断准确率不低于80%

支持不少于5类司法文书数据格式

评查规则

涉及国家秘密的；未成年人犯罪的；以调解方式结案或者确认人民调解协议效力的，但为保护国家利益、社会公共利益、他人合法权益确有必要公开的除外；离婚诉讼或者涉及未成年子女抚养、监护的
